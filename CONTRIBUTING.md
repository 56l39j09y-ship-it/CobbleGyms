# Contributing to CobbleGyms

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository and clone it locally.
2. Install the prerequisites: Java 17+, Gradle.
3. Build the project:
   ```bash
   ./gradlew build
   ```

## Code Style

- Kotlin idiomatic style with 4-space indentation.
- All public classes and functions should have KDoc comments.
- Keep functions focused and small.

## Submitting a Pull Request

1. Create a feature branch: `git checkout -b feature/my-feature`
2. Write your changes with accompanying tests where applicable.
3. Run `./gradlew build` to ensure the project builds cleanly.
4. Open a pull request against `main` with a clear description.

## Reporting Bugs

Open a GitHub Issue with:
- Minecraft/Fabric/Cobblemon version
- CobbleGyms version
- Steps to reproduce
- Expected vs actual behaviour
- Relevant log output

## Feature Requests

Open a GitHub Issue tagged `enhancement` describing the feature and its use-case.
